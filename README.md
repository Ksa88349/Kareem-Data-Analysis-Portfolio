# 📊 Kareem Essam - Data Analysis Portfolio

Welcome to my data analysis portfolio! This repository is designed to showcase my technical skills and strategic mindset as a **Marketing Data Analyst**. 

By combining my background in Computer Science with over 1.5 years of hands-on experience in Media Buying and Performance Marketing, I focus on transforming raw, messy tracking data into clear, profitable business growth pulses.

---

## 🏗️ Repository Architecture

To demonstrate both technical depth and full-scale execution, this portfolio is strictly divided into two core sections:

### 1️⃣ `Tool-Based-Projects`
This directory isolates specific tools within the modern data stack to showcase deep technical proficiency, optimal coding practices, and mastery over individual software features. It is divided by tool:
*   **MySQL:** Advanced querying, database normalization, structural data cleaning, and Exploratory Data Analysis (EDA) using complex CTEs and Window Functions.
*   **Advanced Excel:** Data manipulation, modeling, advanced lookup formulas, and automated data shaping using Power Query.
*   **Power BI:** Transforming relational databases into interactive, dynamic dashboards built around crucial marketing metrics.
*   **Python:** Data manipulation, statistical summaries, and data visualization using specialized libraries like Pandas, NumPy, and Seaborn.

### 2️⃣ `End-to-End-Projects`
This directory houses full-cycle, real-world business scenarios. Instead of working on isolated tasks, these projects mirror the true workflow of a Data Analyst by covering the entire data lifecycle:
*   **Data Extraction & Cleaning:** Sourcing raw, fragmented marketing/e-commerce datasets.
*   **Transformation & Modeling:** Building structured relational models and star schemas.
*   **Dashboarding:** Designing executive-ready dashboards for non-technical stakeholders.
*   **Actionable Advice:** Translating visual patterns into actual business recommendations to optimize ad spend, improve retention, and scale ROI.
